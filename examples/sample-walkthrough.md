# Пример: промпт для инвестиционного отчёта

Полный walkthrough методологии «3 артефакта» — от задачи до готового промпта.

## Задача

Нужно написать квартальный инвестиционный отчёт для партнёров фонда недвижимости. Отчёт должен быть профессиональным, но понятным людям без финансового образования.

---

## Шаг 1. Reference (Образец)

У вас есть прошлогодний отчёт от другого фонда, который произвёл хорошее впечатление на инвесторов. Конвертируем его в markdown и загружаем в чат.

> *Загружаем файл `reference_report.md` в чат*

---

## Шаг 2. Blueprint (Чертёж)

Отправляем AI промпт для анализа:

```
Проанализируй этот reference чтобы потом воссоздать подобное.
Дай краткий, действенный чертёж:
- Что это?
- Тон
- Ключевые паттерны

Уложись в 100 слов. Это твои инструкции для воссоздания подобного текста
без доступа к оригиналу. Всё в codeblock.
```

AI отвечает:

```
BLUEPRINT

Квартальный отчёт инвестфонда недвижимости (8-10 стр.).
Тон: уверенный, но не агрессивный. Факты без приукрашивания.
Структура: (1) Executive Summary — 3 ключевых метрики крупно,
(2) Портфель — таблица объектов с доходностью, (3) Рынок — 2-3
тренда с цифрами, (4) Риски — честно, без замалчивания,
(5) Прогноз — консервативный сценарий + оптимистичный.
Паттерны: каждый раздел начинается с вывода (не с предыстории),
числа визуализированы (таблицы, не абзацы), сравнение с прошлым
кварталом в каждой метрике. Без жаргона — объяснение терминов в скобках.
```

---

## Шаг 3. Success Brief (Бриф успеха)

Заполняем шаблон:

```
SUCCESS BRIEF

Тип и объём: Квартальный инвестиционный отчёт, 8-10 страниц
Реакция получателя: "Понимаю что происходит с моими деньгами,
  фонд работает профессионально, хочу продолжать инвестировать"
НЕ должно звучать как: Сухой банковский отчёт с терминами без
  пояснений. Не должно быть generic AI-формулировок ("в современных
  реалиях", "динамично развивающийся рынок")
Успех это: Партнёры прочитают до конца, не зададут базовых
  вопросов на встрече, двое из пяти увеличат вложения
```

---

```

Структура: (1) Executive Summary — 3 ключевых метрики крупно,
(2) Портфель — таблица объектов с доходностью, (3) Рынок — 2-3
тренда с цифрами, (4) Риски — честно, без замалчивания,

кварталом в каждой метрике. Без жаргона — объяснение терминов в скобках.
###

Вот что нужно для моей версии:

###

Теперь создай план выполнения по шагам (максимум 5).
Определи структуру и задай один вопрос, чтобы начать первый шаг.
```

## Шаг 5. Диалог (коррекция и конвергенция)

После первого ответа AI — работаем по правилам диалога:

**AI** предлагает план из 5 шагов и спрашивает: *«Какие объекты в портфеле и их текущая доходность?»*

**Вы**: "3 объекта: Арамус 42 сотки (доходность 18% годовых), Гарни 28 соток (12%), Цахкадзор 15 соток (продан, +35% за 14 месяцев). Начинай с Executive Summary."

**AI** создаёт первый раздел.

**Вы**: "Метрика 'средняя доходность' неверна — считай взвешенную по объёму инвестиций, а не простое среднее. Исправь и продолжай."

**AI** исправляет и переходит к следующему разделу.

**Вы**: "В разделе Риски добавь: валютный риск (драм/доллар) и риск ликвидности земельных участков. Буллеты, не абзацы."

...

**Вы**: "Теперь создай финальную версию со всеми исправлениями. Один чистый документ."

---

## Результат

Вместо расплывчатого «напиши мне отчёт для инвесторов» — AI получил:
- **Образец** — конкретный пример формата и стиля
- **Чертёж** — что именно делает образец эффективным
- **Бриф** — чёткие критерии успеха для вашей версии
- **Диалог** — итеративные коррекции до финального результата
